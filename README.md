# Rakshak

## Overview

AI-powered tourist safety platform developed for Smart India Hackathon 2025.

## Features

- AI Safety Score
- Geo-Fencing Alerts
- SOS Emergency System
- Police Dashboard
- Blockchain Identity Verification

## Tech Stack

Flutter
React
Node.js
MongoDB
Ethereum
TensorFlow
